# 課題1: C++のclassによるハッシュ表の実現

「C言語（中級）」で与えられている `hash.c` を参考に、次の仕様を満たすハッシュ表を C++ の class で実現しなさい。

## 仕様

`()`、`insert`、`-=` の3つのメンバ関数で、キー（文字列）による**検索**、アイテムの**挿入**、**削除**を行える表を C++ で実現し、振舞いを確認しなさい。

### 操作例

ハッシュ表オブジェクトが `table` とすると、次のように操作するものとする。

| 操作 | 記法 |
|------|------|
| 検索 | `table("hoge")` |
| 挿入 | `table.insert("hoge", x)` |
| 削除 | `table -= "hoge"` |

### 注意

- C++ での文字列比較は、`strcmp` の他、`==` や `s.compare(t)` が使える（`s` と `t` は `string` 型）。

## 加点

- いろいろな型の値を収納できるよう（キーは文字列でよい）に、**テンプレート**にすると **10点プラス**。
