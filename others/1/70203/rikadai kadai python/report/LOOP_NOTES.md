## レポート規程(school-report-tex 準拠、毎周チェック)

- だである調。**考察に感想・疑問文・時系列の経緯話を書かない**
- 図: キャプション下・通し番号・必ず本文から `図 \ref{...}` で参照
- 表: キャプション上・通し番号・必ず本文から参照
- 白黒印刷で判別できる図(線種・マーカー・グレースケールで区別)
- 数式は equation 環境で通し番号
- 本文中の関数名・変数名は `{\tt ...}`
- 引用文献は **2020 年以降**、WebSearch/WebFetch で実在確認してから `sections/06_references.tex` の thebibliography に追加(著者・タイトル・年・URL)。**実在確認できない文献は引用しない**
- 追加実験のソースと実行ダンプは付録(`sections/90_appendix.tex`)に `\verbatiminput` で無加工追加
- 新しい考察は `sections/05_discussion.tex` の小節として追加(構成は「5.x テーマ / 5.x.y 詳細」)
