## Body

Readong about AGI and the ARC-AGI competition, I found out what an ideal AGI would look like:

1. Hypothesis generation: Invent hypothesis for solving a problem, in this case 2d transformations, using prior information (as a scientist would do, from reading books or just mathematical creativity)
2. Hypothesis verification: Explore the hypothesis space of 2d transformations to find the solution to each of the Tasks presented here. This could be done via Monte Carlo or any deterministic algorithm.

It seems to me that this competitions goal is to find a hypothesis space that is not based on elementary 2d transformation but rather using another basis; the orthonormal basis of the Task solutions themselves. This is the same space but a different basis.

Can someone affirm that this is the case?
